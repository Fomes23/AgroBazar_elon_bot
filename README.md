# 🚜 AgroBazar Elon Bot

**AgroBazar** — qishloq xo'jaligi mahsulotlarini (mevalar, sabzavotlar, urug'lar va ko'chatlar) sotuvchilar va xaridorlarni bog'laydigan qulay Telegram bot platformasi.

---

## 🇺🇿 Loyiha haqida (O'zbek tilida)

Ushbu bot dehqonlar va tadbirkorlar uchun o'z mahsulotlarini onlayn e'lon qilish imkonini beradi. Bot orqali mahsulot turi, narxi, miqdori va joylashuvini ko'rsatib e'lon berish mumkin.

### ✨ Asosiy funksiyalar:
* **E'lon berish:** Foydalanuvchilar bir necha qadamda mahsulot rasmi, narxi va tavsifi bilan e'lon joylay oladi.
* **Geolokatsiya:** Sotuvchi mahsulot joylashgan manzilni (lokatsiya) yuborishi mumkin.
* **Kategoriyalar:** Mahsulotlar turlari bo'yicha ajratilgan (Poliz ekinlari, mevalar, urug'lar va h.k.).
* **Admin panel:** Botni boshqarish va e'lonlarni nazorat qilish uchun maxsus bo'lim.
* **Qidiruv va Ko'rish:** Foydalanuvchilar o'z e'lonlarini ko'rishi va boshqarishi mumkin.

---

## 🇬🇧 About the Project (English)

**AgroBazar** is a Telegram bot marketplace designed to connect agricultural sellers and buyers. It simplifies the process of listing and finding farming products.

### ✨ Key Features:
* **Post Ads:** Users can list products with photos, prices, and descriptions.
* **Location Integration:** Sellers can share their exact location for buyers.
* **Product Categories:** Organized listing for fruits, vegetables, seeds, and seedlings.
* **Admin Dashboard:** Tools for managing the bot and monitoring listings.
* **User Management:** Users can view and manage their personal listings easily.

## 🛠 Texnologiyalar (Technologies)

* **Language:** Python 3.x
* **Library:** [aiogram](https://docs.aiogram.dev/) (yoki python-telegram-bot)
* **Database:** SQLite / PostgreSQL
* **Environment:** Dotenv for security

## 🚀 O'rnatish (Installation)

1.  Repozitoriyani klonlang:
    ```bash
    git clone [https://github.com/Fomes23/AgroBazar_elon_bot.git](https://github.com/Fomes23/AgroBazar_elon_bot.git)
    ```
2.  Zarur kutubxonalarni o'rnating:
    ```bash
    pip install -r requirements.txt
    ```
3.  `.env` faylini yarating va bot tokeningizni kiriting:
    ```env
    BOT_TOKEN=sizning_tokeningiz
    ADMIN_ID=sizning_id_ingiz
    ```
4.  Botni ishga tushiring:
    ```bash
    python main.py
    ```
